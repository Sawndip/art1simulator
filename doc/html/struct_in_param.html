Detailed Description

This structure holds every parameters of the network.

The parameters can be set by the user via the "trainart" shell script and are initialized in the io.set_network_values function.

Field Documentation

beta

The beta value is used when testing patterns against clusters it influences the amount of created clusters.

maxPasses

The maxPasses indicates the maximum number of pass that the training will carry out.

minFluc

The minFluc indicates the minimum percentage of fluctuation. If pass ever present a lower fluctuation percentage then the training stage stops.

prefix

The prefix is the string that will be added to output files containing the results.

skip

The skip flag indicates weither the first input file column should be concidered as a class name.

testFile

testFile contains the patterns to test.

testNoise

The testNoise variable indicates the noise percentage to add to the testing patterns.

trainFile

trainFile is the input file: the csv file which is the data set.

trainNoise

The trainNoise variable indicates the noise percentage to add to the training patterns.

vigilance

The vigilance is the more important parameter of the art network. It act as a pattern acceptance threshold for the clusters. The higher the vigilance the pickier the threshold, the higher the amount of clusters.
