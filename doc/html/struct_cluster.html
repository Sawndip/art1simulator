Detailed Description

The structure for a cluster, holding its prototype and assigned patterns.

Field Documentation

inhib

The inhib boolean is a flag indicated whether the cluster is inhibited.

patSet

patSet is a set of patterns ID which belong to the cluster. It means that the previous prot vector has been shaped by these patterns when they where added to the cluster.

prot

The cluster prototype. it's the centroid of the cluster, it's basically a pattern wich contains 1 only where every patterns of the cluster have a 1.
